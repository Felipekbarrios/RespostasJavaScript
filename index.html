Respostas sobre JavaScript

1. JavaScript e Java são a mesma coisa?
Não, JavaScript e Java não são a mesma coisa. JavaScript é uma linguagem de programação usada principalmente para criar interatividade em páginas web, enquanto Java é uma linguagem de programação mais voltada para o desenvolvimento de aplicações corporativas, aplicativos Android, entre outros.
2. Quando surgiu o JavaScript?
O JavaScript foi criado em 1995, por Brendan Eich, enquanto trabalhava na Netscape Communications Corporation.
3. Quem foi o responsável pela criação do JavaScript?
O responsável pela criação do JavaScript foi Brendan Eich. Ele desenvolveu a linguagem em apenas 10 dias para o navegador Netscape Navigator.
4. Qual o caminho trilhado pelo responsável pela criação do JavaScript, trata-se de alguma empresa ainda no mercado?
Brendan Eich foi o cofundador da Mozilla Foundation, uma organização que ainda existe e mantém o desenvolvimento do navegador Mozilla Firefox. Eich também foi o CTO da Mozilla por algum tempo e é uma figura importante no ecossistema de software livre.
